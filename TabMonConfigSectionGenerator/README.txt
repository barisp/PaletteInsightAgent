This project requires the "Configuration Section Designer" extension for Visual Studio. (http://csd.codeplex.com/)

To regenerate TabMonConfigSection.cs and TabMonConfig.xsd, build this project and:
	* Copy the contents of ConfigurationSection.csd.cs into \TabMon\TabMonConfig\TabMonConfigSection.cs
	* Copy the contents of ConfigurationSection.csd.xsd into \TabMon\Resources\TabMonConfig.xsd
